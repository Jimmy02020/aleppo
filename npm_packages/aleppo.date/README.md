# aleppo.date

The [aleppo](https://github.com/Jimmy02020/aleppo) method `date` exported as a [Node.js](https://nodejs.org/) module.

## Installation

Using npm:
```bash
$ npm i aleppo.date
```

In Node.js:
```js
var date = require('aleppo.date');
```

See the [date documentation](https://github.com/Jimmy02020/aleppo#date).
